# 📊 Controle de Peso

Um sistema simples, moderno e responsivo para controle diário de peso, treino e alimentação.

Desenvolvido com HTML, CSS e JavaScript puro, *com foco em uso pessoal* (principalmente no mobile).

---

## 🚀 Funcionalidades

### ⚖️ Controle de Peso
- Visualização e edição do peso atual
- Definição de meta de peso
- Edição direta clicando no número (sem pop-up ou similar)
- Cálculo automático:
  - quanto falta para atingir a meta
  - ou se já passou da meta

---

### 🏋️ Treino Diário
- Adicionar treinos por dia
- Editar e excluir treinos
- Campo de detalhes (exercícios, observações, etc)
- ✔️ Status de treino:
  - bolinha clicável
  - verde = treino feito
  - branco = não feito

---

### 🍽️ Alimentação
- Adicionar refeições
- Editar e excluir
- Definir horário e alimentos
- ✔️ Status de alimentação:
  - controle diário igual ao treino

---

### 💾 Armazenamento
- Dados salvos automaticamente no navegador (localStorage)
- Não foi feito com banco de dados, justamente por ser apenas para uso pessoal
- Se limpar o cache ou trocar de navegador, os dados serão perdidos
- Funciona offline

---

### 📱 Responsividade
- Interface adaptada para celular
- Tabelas viram "cards" no mobile
- Botões maiores para toque
- Layout otimizado para uso diário
